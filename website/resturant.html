Retrieving all the orders placed by the user previously in the program, we calculate the total. This is how. Create placeOrder function. Parameter: dbname Description: This function will allow the user to place their order. Display all the items in the order in a readable format, including the name, quantity, price, and order time. Add them together to print a subtotal. Ask the user if they have a coupon. Ask the user if the coupon is a % off or a specific amount off. Subtract the appropriate amount. Ask the user if they’d like to tip 15%, 18% or 20%. Add New Jersey sales tax of 6.625%. Ask the user if they’d like pickup or delivery. If the user chooses delivery, add $5 to their order.
