In this HTML code, I created a simple photo gallery using just basic HTML. I added a title at the top and displayed five images, each sized at 300 by 200 pixels. The images are loaded from an external source (picsum.photos), and they appear one after another on the page. I kept it clean and straightforward without using any CSS, which makes it a great starting point for learning how to build a gallery with HTML.
