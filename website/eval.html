An online retailer offers tier-based shipping costs for their products. The shipping cost depends on the total purchase amount, the type of product, and the shipping speed selected by the customer. In the main function Create a new menu item named P7 - Problem 7 Ask the user for the total purchase price. Ask the user for the product type Ask the user for their shipping speed Call the function with three arguments. Create a function with three parameters. The function should calculate and return the total shipping cost based on the following conditions: For electronics: If the purchase amount is over $500, shipping is free regardless of shipping speed. Otherwise, 'standard' is $15, 'express' is $35, and 'overnight' is $60. For clothing: Free 'standard' shipping for all purchases. 'Express' shipping costs $20, and 'overnight' costs $40. For books: If the purchase amount is below $50, 'standard' is $5, 'express' is $20, and 'overnight' is $40. For purchases of $50 or more, 'standard' is free, 'express' is $15, and 'overnight' is $30. Print the total amount of the order including shipping. Example: If the order is 300, the product type is electronics, and shipping speed is express, print 335. ________________________________________________________
