One technique commonly used to plan the code is to create a flowchart prior to the code being written As result the entire program follow can be reviewed from beginning to end ahead of time Potential mistakes can be spotted early and corrected before a single line of code is written The oval shape is used to indicate the start or end of a flowchart, marking where the process begins or finishes. Rectangles represent process steps or actions that need to be carried out. Diamonds are decision points where the flow can branch based on yes/no or true/false answers. Parallelograms are used to show input or output, such as entering data or displaying results. Arrows connect these shapes and show the direction of the process flow. ________________________________________________________
