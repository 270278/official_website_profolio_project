Logic Errors Caused by the programmer making a mistake in the algorithm, which makes makes the program to behave unexpectedly Syntax error Typos or writes some code (or forgets to write some code) that doesn’t follow the rules of the language Like colon/semicolon Parenthesis Curly braces Indentation Quotes Variable definition Run time error When program fails in the midst of running Referred to as a “bug” The key to identifying the error is that the program runs/compiles initially, but fails and cannot complete its processes A run time error may result from: Dividing by 0- if done it repeats the number for infinity Inappropriately entered data type Many other advanced possibilities Overflow error When a program is required a calculation that is outside of defined range of values Due to memory allocation constraints that programming languages require, certain values are too big to calculate and/or display Semantic errors happen when your code runs without crashing but doesn’t do what you intended ________________________________________________________
