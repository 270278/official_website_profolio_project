Most humans see and use objects Computers, cars, smartphones, televisions, etc. All objects have properties such as size, shape, color, etc. Objects can be used for different things. Abstraction is an approach used in object-oriented programming languages such as C++ and Java In this process, most data is hidden from view in order to make the object easier and more efficient to use E.g., Smartphones, Tablets, etc. ________________________________________________________
